## Description

Every AI agent eventually needs to ask the user a question.

Onboarding. Discovery. Coaching. Matchmaking. Assessment.

And they all ask the same five boring questions:
- "How can I help you?"
- "What are your goals?"
- "Tell me about yourself."

**Dead questions get dead answers.**

BetterAsk is an API built on 607 battle-tested questions from the book END SMALL TALK by Cory Stout — organized into 12 psychological archetypes that extract real signal from humans.

### How it works

```
POST /generate
{
  "context": "onboarding",
  "about": "what motivates this user",
  "depth": "medium"
}
```

You get back a structured prompt for your LLM — archetype, generation instructions, and a real example from the corpus. Pass it to your model. Get a question that makes your user lean in instead of check out.

### Score any question

```
POST /score
{
  "question": "What's the most expensive thing you own that you never use?"
}
```

Returns a 6-dimension quality score: Surprise, Specificity, Conversation Fuel, Self-Revelation, Fun Factor, Universality. Now you know if your question is a 3 or an 8 before your users ever see it.

### The 12 Archetypes

Every great question follows a pattern:

| # | Archetype | What it does |
|---|-----------|-------------|
| 1 | 🔄 Reframe | Familiar thing through unexpected metaphor |
| 2 | 🎯 Specificity Trap | Precise number that reveals personality |
| 3 | ⚖️ False Binary | Two good options — your pick says everything |
| 4 | 🪞 Mirror | See yourself from the outside |
| 5 | 🧪 Thought Experiment | Absurd hypothetical that's secretly revealing |
| 6 | 🕰️ Time Machine | Memory as a lens on life |
| 7 | 📈 Absurd Escalation | Normal → ridiculous |
| 8 | 💔 Vulnerability Door | Gentle emotional opening |
| 9 | 🏷️ Identity Sort | Self-categorization |
| 10 | 🧠 Explain-It Test | Expose the knowledge-confidence gap |
| 11 | 🌍 World-Builder | Design your ideal |
| 12 | ⛓️ Chain | Rapid sequence that builds trust through rhythm |

## First Comment (post immediately after launch)

Quick start in 30 seconds:

```bash
# Generate a question — no signup, no API key, just curl it
curl -X POST https://betterask.dev/generate \
  -H "Content-Type: application/json" \
  -d '{"context":"onboarding","about":"what drives this person","count":1}'

# Score any question
curl -X POST https://betterask.dev/score \
  -H "Content-Type: application/json" \
  -d '{"question":"If you repeated yesterday 365 times, where would you be?"}'

# Browse all 6 archetypes
curl https://betterask.dev/archetypes
```

Full docs at https://betterask.dev/docs

Happy to answer any questions about the methodology or API design. 🤙
